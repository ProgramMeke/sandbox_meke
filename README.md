# mekemap   

mekemap is a Python library for building an interactive map in which NER annotations are linked to their coordinates

## Installation

Use the package manager [pip](https://pip.pypa.io/en/stable/) to install mekemap

```bash
pip install mekemap
```

## Usage

```python
import mekemap

# initializes a map
mekemap.map()

```
